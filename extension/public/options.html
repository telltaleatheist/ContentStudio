ContentStudio connection

The localhost port ContentStudio's ingest server listens on (default 43117). Shown on ContentStudio's Analytics page.
Paste the bearer token from ContentStudio's Analytics page.

Channels (from ContentStudio)

Channels are pulled live from ContentStudio — the app already has them registered, so there is nothing to configure here. Save the port and token above, then Refresh.
